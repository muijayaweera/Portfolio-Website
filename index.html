Project 1 - Ostocare

A mobile application developed using Flutter and Firebase to support ostomy patients in Sri Lanka. The app helps users locate specialized hospitals, contact stoma care nurses, track stoma bag changes, access resources, and record recovery progress. It also includes a settings page, awareness features, and data access for healthcare providers to improve patient care and teleconsultation.

Project 2 - IoT Based Smart Shelf

A real-time inventory monitoring system built using ESP32, Arduino, load cells, RFID modules, and Firebase. The system displays product weights, identifies stock levels, and visualizes data through a connected dashboard. Designed to improve stock accuracy and efficiency in retail environments, the system features LCD displays, alerts, and a read-only web interface for inventory insights.
